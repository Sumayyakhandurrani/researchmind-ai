# researchmind-ai

An AI-powered Research Assistant that enables semantic understanding of research papers using Retrieval-Augmented Generation (RAG).

## Features (In Progress)

- PDF ingestion and parsing
- Text chunking with overlap
- Sentence embedding generation
- FAISS vector database for semantic search (in progress)

## Tech Stack

- Python
- FastAPI (planned)
- Sentence Transformers
- FAISS
- PyMuPDF

## Project Goal

To build an intelligent system that can:
- Summarize research papers
- Compare multiple papers
- Extract methodologies
- Identify research gaps
- Assist in literature review writing
